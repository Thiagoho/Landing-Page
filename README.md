# Bella Nails - Landing Page

## Sobre o Projeto

Este projeto consiste no desenvolvimento de uma Landing Page para uma profissional da área de manicure.

O objetivo é oferecer uma presença digital profissional, permitindo que clientes conheçam os serviços oferecidos, visualizem trabalhos realizados e entrem em contato de forma rápida através do WhatsApp.

## Funcionalidades

* Página inicial moderna
* Apresentação dos serviços
* Galeria de fotos
* Seção "Sobre Mim"
* Informações de contato
* Integração com WhatsApp
* Layout responsivo para celular, tablet e desktop

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Figma (Protótipo)

## Objetivo

Criar uma página profissional para auxiliar na divulgação do negócio e melhorar a experiência dos clientes durante o processo de contato e agendamento.
